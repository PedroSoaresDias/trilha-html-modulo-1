História dos computadores

Hitória dos computadores: Na década 40 do século 20, os computadores eram de grande porte, são barulhentos, quando esquentava, atraia os insetos e entravam dentro dos computadores e teve origem da palavra Bug quando os insetos entrava neles.
Após anos de evolução, os computadores reduziram de tamanho e melhoraram a sua eficiência com o passar do tempo, as primeiras versões eram maiores e mais barulhentas e nos dias atuais já são pequenos e silenciosos, e são utilizados para pelas empresas de pequeno e grande porte, e também para o uso doméstico.

O que são Clients?

Lado do Cliente: O lado do cliente para a web se refere a como o usuário interage com a internet, para o desenvolvimento web é chamado de Front-end, que é como o usuário pode ver e interagir na internet.
No desenvolvimento front-end tem a parte de UI interface de usuário como ele vai interagir, tem questão da UX experiência de usuário para que tenha a melhor experiência possível, a parte de acessibilidade para os usuários que têm necessidades de usar leitor de tela ou outras necessidades, e a parte de responsividade para o website se adequar em diferentes tamanhos de telas.

O que são Servers?

Lado do Servidor: O lado do servidor para a web mostra funciona um website ou uma aplicação web onde o usuário não consegue ver ou interagir, o desenvolvimento web para o lado de servidor é chamado de Back-end, que é como funciona um site por baixo dos panos.
No desenvolvimento back-end tem a parte de regras de negócios, fazer interação com o banco de dados, criação de APIs (Application Programming Interface) seja para fazer a comunicação com o banco de dados ou para ser consumida no front-end e fazer a comunicação entre o back-end e o front-end.

Como é uma estrutura básica de um HTML?

Estrutura do HTML: A linguagem HTML não é uma linguagem de programação, ela é uma linguagem de marcação, o HTML é utilizado para desenvolver websites de forma estruturada para passar o conteúdo do site para a internet.
O desenvolvimento de um site escrito em HTML, para desenvolver, é preciso seguir a semântica do HTML de forma correta para evitar problemas como a falta de acessibilidade que pode dificultar para quem usa os leitores de tela, terá problemas com o SEO (Search Engine Optimization) para ter a melhor posição nos motores de busca como o Google e o Bing.

Entendendo a diferença entre listas ordenadas e não ordenadas

Lista ordenada e não ordenada: A lista ordenada é uma forma de fazer uma lista de forma numerada, se tiver um site de receitas pode ser útil para fazer a ordem para fazer alguma receita passo a passo.
Já a lista não ordenada é utilizada para definir os tópicos de um determinado assunto, o que diferencia da lista ordenada para não ordenada, é que na não ordenada não utiliza os números para definir os tópicos.
