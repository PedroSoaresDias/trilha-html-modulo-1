História dos computadores

Hitória dos computadores: Na década 40 do século 20, os computadores eram de grande porte, são barulhentos, quando esquentava, atraia os insetos e entravam dentro dos computadores e teve origem da palavra Bug quando os insetos entrava neles.
Após anos de evolução, os computadores reduziram de tamanho e melhoraram a sua eficiência com o passar do tempo, as primeiras versões eram maiores e mais barulhentas e nos dias atuais já são pequenos e silenciosos, e são utilizados para pelas empresas de pequeno e grande porte, e também para o uso doméstico.

O que são Clients?

Lado do Cliente: O lado do cliente para o web se refere a como o usuário interage com a internet, para o desenvolvimento web é chamado de front-end, que é como o usuário pode ver e interagir na internet.
No desenvolvimento front-end tem a parte de UI interface de usuário como ele vai interagir, tem questão da UX experiência de usuário para que tenha a melhor experiência possível, a parte de acessibilidade para os usuários que têm necessidades de usar leitor de tela ou outras necessidades, e a parte de responsividade para o website se adequar em diferentes tamanhos de telas.
